## 3. 推荐测试问题

事实型：

```text
什么是动作电位？
什么是炎症？
什么是细胞损伤？
```

机制型：

```text
动作电位的产生机制是什么？
缺氧会导致哪些病理生理变化？
炎症反应的基本过程是什么？
```

比较型：

```text
病理学和病理生理学对炎症的解释有什么不同？
免疫应答和炎症反应有什么关系？
```

跨教材型：

```text
哪些教材都涉及细胞损伤相关知识？
传染病学中的感染过程和病理学中的炎症反应如何关联？
```
